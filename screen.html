tone3000 API key

            

                The tone3000.com REST API requires authentication. Without a key,
                the plugin runs in deep-link mode (buttons that open tone3000 in
                your browser). With a key, candidates are listed in-app and the
                batch auto-download mode is unlocked.

Extract Rocksmith IRs

            

                Pulls cab IRs out of the Wwise .bnk
                files inside your gears.psarc and
                saves them as 48 kHz float32 .wav
                files in nam_irs/rocksmith/. The
                auto-mode batch prefers these over tone3000 for cabs.
