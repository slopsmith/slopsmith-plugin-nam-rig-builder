Global FX layered around every tone of every song. Pre runs first (sees the raw DI — typical place for input gate / compressor / tuner). Post runs after the cab IR (typical place for limiter / room reverb / headphone EQ). Each piece supports the same Library / VST / Bypass / reorder controls as a song chain. Signal flow: guitar → master_pre → song → master_post → output.

How to use: (1) ＋ Add piece → pick a Rocksmith effect OR a VST plug-in. VST pieces are ready to use immediately — just hit 🎛 Edit VST to tweak its parameters. Rocksmith pieces start unassigned: click Assign… then pick a NAM/IR from your library or paste a VST plug-in path. (2) Master pieces are global — they don't appear on the Gear tab.
🎸 AMP must be on for master to apply. If you flip the AMP button mid-song, Rig Builder now auto-reloads the chain (including master pre/post) ~1 second later — no need to leave and re-enter. Disable with window.__rbAmpAutoApply = false if it ever misfires.

VST / Audio Unit plugins

            

                Scan your installed VST3 / Audio Unit plugins once here — the
                results feed the 📚 Library picker that appears on every gear.
                Scanning instantiates each plugin to validate it, so it can take
                up to a minute; a broken plugin may make it time out (check the
                console for the offending path).
            

            

                
                
            

        


        

            
Auto-mode policy

            
            
        


        

            

                Chain preloader DEFAULT
            

            

                Pre-loads every tone of the song into one big chain at song start,
                then switches between tones by toggling bypass instead of reloading
                NAMs. Eliminates the tone-change spike
                completely. Steady-state memory is higher (every NAM stays
                loaded) but on M-series Macs the trade-off is clear-cut.
            

            

                Turn it off if you're on a weak x86 laptop and hear crackling, or if
                a specific song is misbehaving — playback then falls back to the
                cooperative load/unload path. Kill-switch in DevTools:
                window.__rbMegaChain = false.
