            Global FX layered around every tone of every song. Pre
            runs first (sees the raw DI — typical place for input gate / compressor / tuner).
            Post runs after the cab IR (typical place for limiter / room
            reverb / headphone EQ). Each piece supports the same Library / VST / Bypass / reorder
            controls as a song chain. Signal flow:
            guitar → master_pre → song → master_post → output.
        

        

            How to use:
            (1) ＋ Add piece → pick a Rocksmith effect OR a VST plug-in.
            VST pieces are ready to use immediately —
            just hit 🎛 Edit VST to tweak its parameters.
            Rocksmith pieces start unassigned: click Assign…
            then pick a NAM/IR from your library or paste a VST plug-in path.
            (2) Master pieces are global — they don't appear on the Pending or Gear catalog tabs.
        

        

            ⚠️ Important: The AMP button on the song
            player must be ON before you load the song for the master chain (and per-song chains)
            to apply during playback. If you forgot, the workaround is: turn AMP on → leave the song → re-enter
            the song. We're working on auto-applying when AMP toggles mid-song.

tone3000 API key

            

                The tone3000.com REST API requires authentication. Without a key,
                the plugin runs in deep-link mode (buttons that open tone3000 in
                your browser). With a key, candidates are listed in-app and the
                batch auto-download mode is unlocked.
            

            

                
                
            

            

        


        

            
VST / Audio Unit plugins

            

                Scan your installed VST3 / Audio Unit plugins once here — the
                results feed the 📚 Library picker (Plugins tab) on every gear.
                Scanning instantiates each plugin to validate it, so it can take
                up to a minute; a broken plugin may make it time out (check the
                console for the offending path).

Extract Rocksmith IRs

            

                Pulls cab IRs out of the Wwise .bnk
                files inside your gears.psarc and
                saves them as 48 kHz float32 .wav
                files in nam_irs/rocksmith/. The
                auto-mode batch prefers these over tone3000 for cabs.
            

            

                
                
            

            

        


        

            
Export my mappings as defaults

            

                Snapshot your current gear → tone3000 capture choices into
                default_captures.json (shipped
                with the plugin). A fresh install then auto-downloads these exact
                captures during batch / auto-download instead of searching tone3000.
