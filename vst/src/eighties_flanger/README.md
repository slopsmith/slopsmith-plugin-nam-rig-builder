# EightiesFlanger

Bundled DPF VST3 for Rocksmith `Pedal_80sFlanger`.

The local reference is `pedals/80s flanger.jpg`, an MXR Flanger Original
schematic with SAD1024 BBD delay, Regen, Sweep, Manual, and Width controls.
Rocksmith exposes only `Rate`, `Depth`, and `Mix`, so Regen and Manual/Sweep
center are voiced internally for an aggressive 80s jet sweep.
