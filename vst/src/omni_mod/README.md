# OmniMod

Bundled DPF VST3 for Rocksmith `Pedal_OmniMod`.

The local references are `pedals/omnimod_1.gif` and `pedals/omnimod_2.jpg`,
both pointing at a Uni-Vibe/Shin-ei style photocell phase modulation circuit.
Rocksmith exposes `Rate`, `Depth`, and `Mix`; lamp lag and photocell response
are voiced internally.
