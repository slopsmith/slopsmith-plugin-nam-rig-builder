# ModernFlanger

Bundled Rocksmith `Pedal_ModernFlanger` model.

Reference: `pedals/modern flange.pdf`, a Moog MF-108M Cluster Flux schematic
with LM13700 mix/feedback VCAs, sixth-order Chebyshev input/output filters, and
MN3009/MN3007-style BBD delay elements. Rocksmith exposes `Rate`, `Depth`,
`Regen`, and `Mix`.
