# PlanePhase

Bundled DPF VST3 for Rocksmith `Pedal_PlanePhase`.

The local reference is `pedals/plane phase.gif`, an AP-7 style phaser with a
distortion input driver, eight phase-converter stages, Resonance, Jet Level,
and Rate controls. Rocksmith exposes only `Rate`, `Depth`, and `Mix`, so
Resonance and Jet Level are voiced internally.
