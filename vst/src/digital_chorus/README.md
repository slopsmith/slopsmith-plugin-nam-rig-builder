# DigitalChorus

Bundled Rocksmith `Pedal_DigitalChorus` model.

Reference: `pedals/digital chorus.png`, a digital chorus around an ESS delay
chip with `Rate`, `Depth`, `Filter Low`, `Filter High`, and effect level. The
plugin keeps the same Rocksmith controls: `Rate`, `Depth`, `LoFilter`,
`HiFilter`, and `Mix`.
