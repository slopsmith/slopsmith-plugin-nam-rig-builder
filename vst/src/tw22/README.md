# TW22

Bundled VST3 amp for Rocksmith `Amp_TW22`, based on the local Fender
Super-Sonic 22 schematic.

Rocksmith exposes only `Gain`, `Bass`, `Mid`, and `Treble`. The plugin keeps
that contract:

- `Gain` blends from the Vintage/Fat clean preamp behavior into the Burn
  channel's cascaded gain staging.
- `Bass`, `Mid`, and `Treble` drive the shared Fender-style tone stack.
