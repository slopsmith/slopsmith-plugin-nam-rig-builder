# DigitalVerb

Bundled VST3 for Rocksmith `Pedal_DigitalVerb`.

- Reference: `/Users/nacho/Files/slopsmith/pedals/digital verb.png`
- Circuit target: Boss RV-2 style early digital reverb
- Rocksmith knobs: `Time`, `Mix`, `Depth`, `Tone`

The DSP keeps the dry path clean and blends in a stereo algorithmic tail.
`Tone` controls damping/brightness, `Depth` controls room depth, pre-delay
and density, and `Time` controls the decay feedback.
