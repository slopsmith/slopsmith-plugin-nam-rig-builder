# DSL100

Bundled RigBuilder amp for Rocksmith's `Amp_MarshallDSL100H`.

Local references:

- `amps/Marshall DSL100/JCM2-60-02 (2003) iss9.pdf`
- `amps/Marshall DSL100/JCM2-61-00 (2001) iss5.pdf`
- `amps/Marshall DSL100/JCM2-62-02 (2001) iss3.pdf`
- `amps/Marshall DSL100/DSL50-100 manual (2004).pdf`

The plugin models the JCM2000 DSL100 as one Rocksmith amp: low `Gain` is
Classic Clean, mid `Gain` is Classic Crunch / 2203-style drive, and high
`Gain` is Ultra OD1. `Res` maps to the deep/resonance low-feedback behavior,
while `Pres` maps to the upper feedback presence and speaker bite.
