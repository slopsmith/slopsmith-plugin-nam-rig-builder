# ClassicFlanger

Bundled DPF VST3 for Rocksmith `Pedal_ClassicFlanger`.

The local reference is `pedals/classic flanger.pdf`, a Boss BF-2 schematic.
That circuit has Manual, Depth, Rate, and Resonance around an MN3207 BBD.
Rocksmith exposes only `Rate`, `Depth`, and `Mix`, so Manual and Resonance are
voiced internally for a darker, smoother analog flanger.
