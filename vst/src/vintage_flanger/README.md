# VintageFlanger

Bundled Rocksmith `Pedal_VintageFlanger` model.

Reference: `pedals/vintage flanger.jpg`, an Electro-Harmonix Deluxe Electric
Mistress schematic with RD5106A BBD delay, LM324 LFO/range control, and a
fixed Color/feedback path. Rocksmith exposes `Rate`, `Depth`, and `Mix`; Color
is voiced internally.
