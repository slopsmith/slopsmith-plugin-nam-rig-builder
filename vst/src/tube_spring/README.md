# TubeSpring

Bundled VST3 for Rocksmith `Pedal_TubeSpring`.

- Reference: `/Users/nacho/Files/slopsmith/pedals/spring reverb.gif`
- Circuit target: high-voltage spring tank driver/recovery reverb
- Rocksmith knobs: `Mix`, `Depth`

`Depth` controls dwell, tank decay, and spring drip. `Mix` is the blend.
The dry path is not clipped or boosted; the analog/tube character is applied
inside the virtual tank drive.
