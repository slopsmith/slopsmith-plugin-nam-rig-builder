# ModDelay

Bundled VST3 for Rocksmith `Pedal_ModDelay`.

Reference: local `pedals/moddelay_1.gif` and `pedals/moddelay_2.gif`, Ibanez
DLL10 schematics used as a practical stand-in for the Rocksmith DML10-inspired
pedal.

Implemented controls match Rocksmith:

- `Time`: delay time, mapped from 20-900 ms.
- `Feedback`: regen/feedback.
- `Mix`: dry/wet balance.
- `Rate`: modulation speed, mapped from 0-3.5 Hz.
- `Depth`: modulation width.
